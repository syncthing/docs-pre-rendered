GET /rest/system/version
========================

Returns the current Syncthing version information.

.. code-block:: json

    {
      "arch": "amd64",
      "longVersion": "syncthing v0.10.27+3-gea8c3de (go1.4 darwin-amd64 default) jb@syno 2015-03-16 11:01:29 UTC",
      "os": "darwin",
      "version": "v0.10.27+3-gea8c3de"
    }
