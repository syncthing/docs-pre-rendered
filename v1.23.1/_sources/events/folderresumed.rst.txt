FolderResumed
-------------

Generated when the configuration changes regarding the "paused" state
of a folder.  Sent repeatedly for each changed folder.

.. code-block:: json

    {
      "id": 93,
      "globalID": 93,
      "type": "FolderResumed",
      "time": "2021-06-07T23:45:03.414609034+02:00",
      "data": {
	"id": "abcde-fghij",
	"label": "My folder"
      }
    }
